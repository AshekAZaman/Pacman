# Pacman

This program runs the game pacman which moves around the canvas, controlled by keyboard using the arrow keys for suitable directions. 
Once the pacman touches the edge of the canvas or the enemy the game is over. 
Learn more about the game at https://en.wikipedia.org/wiki/Pac-Man

To run the programming you need to have processing installed in your computer. Run Pacman.pde in 
your computer.
